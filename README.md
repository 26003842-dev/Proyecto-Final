Funcionamiento del programa

Este programa realiza cotizaciones automáticas para la empresa de seguros TK-U.
Su objetivo principal es calcular el precio final de un seguro a partir de un precio base y distintos recargos, según las características del asegurado.

El programa permite realizar múltiples cotizaciones, repitiendo el proceso hasta que el usuario escriba la palabra SALIR.

Primero, el programa solicita los datos básicos del asegurado.
Luego pide datos más específicos que afectan el cálculo de los recargos, como la edad, si está casado, la edad del cónyuge, si tiene hijos, cuántos hijos tiene, la cantidad de propiedades y el salario del asegurado.

Cada uno de estos datos tiene un porcentaje de recargo asignado, el cual se suma al precio base para obtener el precio final de la cotización. Algunos de los recargos se calculan utilizando rangos, especialmente los relacionados con la edad.

El programa permite continuar realizando cotizaciones de forma consecutiva hasta que el usuario decida salir.

Consideraciones a mejorar

Validar que el usuario ingrese solo números cuando se le solicita la edad o la cantidad de hijos.

Evitar que el usuario ingrese edades negativas o menores a 18 años.

Mostrar mensajes de error cuando el usuario deje campos vacíos.

Permitir corregir un dato sin tener que repetir toda la cotización.
